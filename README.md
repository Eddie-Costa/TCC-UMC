# 🌍 Sistema Inteligente de Planejamento de Viagens

## 📖 Descrição
Este projeto tem como objetivo desenvolver um site de viagens capaz de auxiliar usuários no planejamento completo de suas viagens.
A plataforma fornece recomendações personalizadas de destinos, melhores períodos para viajar, preços médios e dicas úteis, facilitando a tomada de decisão do usuário.

## 🎯 Objetivo
Criar uma aplicação web que automatize o planejamento de viagens, oferecendo informações relevantes como clima, custo e sazonalidade, permitindo que o usuário organize sua viagem de forma prática, rápida e eficiente.

## 🚀 Funcionalidades
* Sugestão de destinos com base em preferências do usuário
* Indicação da melhor época para viajar
* Exibição de preços médios (passagens, hospedagem, etc.)
* Informações sobre clima e estações do ano
* Dicas de pontos turísticos e atividades
* Planejamento básico de roteiro de viagem

## 🛠️ Tecnologias utilizadas
* Java
* SQL
* MySQL
* Spring Framework
